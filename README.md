Project Name: EcoAudit - Community Waste Logger

Description:
EcoAudit is a web application that allows users to log waste disposal entries with category, weight, and automatically validated geolocation.

Tech Stack:
HTML, CSS, JavaScript, Firebase Firestore, Leaflet Map API

Features:
- Captcha-based access screen
- Waste entry form
- Browser geolocation validation
- Firebase data persistence
- Real-time dashboard
- Total waste calculation
- Map visualization
- PDF export

How to Run:
Open index.html using Live Server.

Deployment:
Hosted on Netlify/Vercel.

Important:
Location permission must be allowed for waste logging.
